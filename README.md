This repository contains a collection of Python Jupyter Notebooks and documents focused on mastering **Dictionaries**. The goal of this task was to explore different dictionary lengths, nesting levels, and memory management behaviors.

### 🚀 What's Inside?
## **1. Dictionary Varieties:**

- Examples of dictionaries with lengths ranging from 5 to 15+ keys.

- Real-world data structures like Employee Records, Flight Tickets, and Product Specs.

## **2. Nested Data:**

- Implementation of dictionaries containing Lists and Nested Dictionaries.

- Practice on "drilling down" into multi-level data.

## **3. Core Methods Explored:**

- **update():** Merging data directly into memory.

- **clear():** Wiping a dictionary in-place.

- **get() & setdefault():** Safe way to access and insert keys.

- **keys()**, **values()**, and **items():** Understanding dictionary "Views."

## 🧠 **Key Concepts Learned**
- **In-Place vs. Return Values:** Understanding why **dict.update()** and **dict.clear()** returns `None` while **dict.values()** returns `data`.

- **Memory Management:** How Python modifies objects in "Main Memory" (Mutable vs. Immutable).

- **Git Workflow:** Initializing a repository, handling remote rejections, and syncing local code with GitHub.
